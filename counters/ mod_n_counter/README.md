# Mod-N Counter

## Description
- Parameterized Mod-N counter
- Counts from 0 to N-1
- Wrap-around on reaching N-1
- Enable controlled
- Asynchronous active-low reset

## Parameters
- WIDTH : Counter bit width
- N     : Modulus value

## Files
- mod_n_counter.sv – RTL design
- tb_mod_n_counter.sv – Testbench
